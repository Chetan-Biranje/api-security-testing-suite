# API Security Testing Suite

Comprehensive API security testing tools and frameworks for penetration testers and security engineers.

## Description
Automated and manual API security testing suite covering REST, GraphQL, SOAP, and WebSocket protocols.

## Topics
api-security, penetration-testing, security-testing, automation, tools

## Features
- REST API Security Testing
- GraphQL Vulnerability Detection
- OAuth/JWT Analysis
- Rate Limiting Tests
- CORS Misconfiguration Detection

## Installation
```bash
git clone https://github.com/Chetan-Biranje/api-security-testing-suite
cd api-security-testing-suite
pip install -r requirements.txt
```
